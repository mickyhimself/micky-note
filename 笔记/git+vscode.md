# git+vscode
1. 右键文件夹打开终端，输入`git stutus`
2. 工作区--资源管理器（vscode第一个选项）
3. 暂存区--源代码管理(vscode第三个选项)
此处可以用+将修改加入到暂存区中
```ctrl+ ` ```:打开终端，
`git status`:查看暂存区的情况，`git add 文件名`:将文件加入暂存区
`git rm --cached 文件名`:将文件从暂存区删除
